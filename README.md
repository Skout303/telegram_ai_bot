# 🤖 Telegram AI Bot

Этот проект — умный Telegram-бот с памятью 🧠, который использует **OpenAI GPT-4o-mini**.  
Он может вести связные диалоги, помнить контекст, и работать 24/7 (на сервере Render или Railway).

---

## 🚀 Возможности

- 💬 Понимает и отвечает на сообщения в Telegram  
- 🧠 Запоминает контекст общения (сохраняется в `memory.json`)  
- 🧹 Команда `/reset` очищает память  
- ☁️ Может работать 24/7 на бесплатном хостинге (Render, Railway)
